# US Healthcare RCM Billing Sandbox

Interactive Revenue Cycle Management simulator for healthcare billing professionals.

## Features

- CMS-1500 Professional Claims
- UB-04 Institutional Claims
- Reimbursement Calculator
- Denial Management Simulator
- KPI Dashboard
- RCM Terminology Glossary
